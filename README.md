### Projetos em Destaque

#### [Alert.ia](https://github.com/Drecko-Vibe/AlertIA)
Extensão (Manifest V3) desenvolvida para otimizar a triagem de atendimentos em plataformas de Service Desk. Utiliza a API do Google Gemini para interceptar requisições de rede (XHR/Fetch) e sugerir transferências de tickets com base na análise semântica do chat, reduzindo significativamente o tempo de resposta e falhas de direcionamento.

#### [LUMA 2.0 - W Command Center](https://github.com/Drecko-Vibe/LUMA)
Centro de comando inteligente criado para otimizar o fluxo de trabalho de suporte técnico no ecossistema Clinicorp. O projeto conta com injeção de interface customizada (UI com *glassmorphism*), motor de inteligência artificial próprio (TagSense) e sistema de varredura de protocolos em lote.
